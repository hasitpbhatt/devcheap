# Prompt Templates — DevCheap

A small collection of practical prompt templates you can adapt.

## 1) Code repair (PR-ready)

```
You are an assistant that proposes minimal code changes to fix failing tests. Provide a unified diff with the smallest change that passes the test and a short explanation.
```

## 2) Refactor to async

```
Refactor the following synchronous function to async/await while keeping the same behavior and edge-case handling. Show only the refactored function.
```

## 3) Generate unit tests

```
Generate unit tests for the following function. Use Jest and include edge cases.
```

Contribute new templates by opening an issue or a PR.
