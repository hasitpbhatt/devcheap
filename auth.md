# DevCheap Agent Authentication

DevCheap does not require authentication for browsing or searching deals.

## Public Access

All deal browsing, searching, and filtering capabilities are available without authentication.

## API Access

The deals database is publicly available at:

- `https://devcheap.click/data/deals.jsonl`

No API key or authentication is required for read access.
